# Anamnesis Nutricional - Camila Lagos

Formulario interactivo de anamnesis nutricional para primera consulta de nutrición integrativa.

## Características

- ✅ Formulario de 8 secciones con progreso visual
- ✅ Validación de campos requeridos
- ✅ Interfaz responsiva (móvil y desktop)
- ✅ Integración con Supabase para almacenamiento
- ✅ Emails automáticos a nutricionista y paciente
- ✅ Branding verde/coral profesional

## Cómo usar

Compartir el link con pacientes para que completen la anamnesis.

## Privacidad

Los datos se utilizan exclusivamente para evaluación clínica.
